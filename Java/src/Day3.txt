1.iterator
컬렉션에 저장된 요소를 읽어오는 방법을 Iterator 인터페이스로 표준화하고 있습니다.

2.프로세스(process)란 단순히 실행 중인 프로그램(program)이라고 할 수 있습니다.
즉, 사용자가 작성한 프로그램이 운영체제에 의해 메모리 공간을 할당받아 실행 중인 것을 말합니다.

3.스레드
스레드(thread)란 프로세스(process) 내에서 실제로 작업을 수행하는 주체를 의미합니다.
모든 프로세스에는 한 개 이상의 스레드가 존재하여 작업을 수행합니다.

4.절대경로와 상대경로
파일 IO등을 이용하게되면
절대경로보다 상대경로를 더 많이 쓰게됨
/-> root
./->현재폴더
../->현재위치의 상위폴더로가겠음
